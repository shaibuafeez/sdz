## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/SeventhOdyssey71/sudoz-dog.git
   cd sudoz-dog
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
